# Weather Data Analysis and Machine Learning

This project analyzes weather data from **Dhaka** and **Delhi** (01-May-2019 to 30-April-2024) using data science and machine learning techniques. It includes comprehensive **Exploratory Data Analysis (EDA)**, **regression models** for predicting temperature and rainfall, and **classification models** for identifying the city based on weather features.

## Features
- Comprehensive Exploratory Data Analysis (EDA)
- Data preprocessing and feature engineering
- Temperature and rainfall prediction using Regression Models
- City classification using Logistic Regression and Support Vector Machine (SVM)
- Model evaluation and performance comparison

## Technologies Used
- Python
- Pandas & NumPy
- Matplotlib & Seaborn
- Scikit-learn
- Google Colab / Jupyter Notebook

## Regression Models
- Linear Regression
- Ridge Regression
- Lasso Regression
- Elastic Net
- Random Forest Regressor

## Classification Models
- Logistic Regression
- Support Vector Machine (SVM)

## Dataset
Two weather datasets containing daily weather records for **Dhaka** and **Delhi** from **May 2019 to April 2024**.

## Outcome
The project compares multiple machine learning models for weather prediction and city classification, providing insights into weather patterns and model performance.
